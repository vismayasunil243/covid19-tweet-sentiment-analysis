# COVID-19 Tweet Sentiment Analysis

This project uses Natural Language Processing (NLP) and machine learning to classify COVID-19 related tweets as positive or negative.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLP

## Techniques Used
- Text preprocessing
- Stopword removal
- TF-IDF Vectorization
- Machine learning classification

## Models Used
- Logistic Regression
- DecisionTreeClassifier
- Random Forest

## Evaluation Metrics
- Accuracy Score
- Classification Report
